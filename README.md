<h1>飲水補給站</h1>

飲水補給站是協助使用者尋找到＂台北市內指定位置附近的直飲台＂的 line-bot。

<h2>使用說明 :</h2>
1. 透過下方 QRcode 或輸入 ID，與飲水補給站成為好友。
<br>
2. 按下左下角的 "+" 圖形，點選 "位置資訊"。
<br>
3. 點擊畫面深色地址一下，將目前的定位送出。 ( 若不是要使用目前位置，也可以使用搜尋功能查詢目的地，搜尋完成後再點擊深色地址送出。)
<br>
4. 靜待 3 秒，即可收到指定位置附近的直飲台。
<br>
<br>

- 若等待太久可以再輸入一次訊息喚醒機器人。
